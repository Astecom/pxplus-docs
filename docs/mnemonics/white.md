# Mnemonics

**'WHITE' & '_WHITE'** |  **_Colour Text_**  
---|---  
  
**GUI Display/Printer _or_ Character Display**

##  Formats

_Foreground:_**'WHITE'**  
  
 _Background:_**'_WHITE'**

##  Description

All input or output following this mnemonic will be in White foreground or background.
