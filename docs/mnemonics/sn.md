# Mnemonics

**'SN'** |  **_Native Screen Mode_**  
---|---  
  
**Behaviour**

##  Description

**_(Default)_** When you use **'SN'** , the "clear" mnemonics fill the cleared regions with the current colour, and the **'RM'** mnemonic resets the output to the default colours.
