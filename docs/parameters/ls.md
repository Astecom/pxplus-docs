# System Parameters

**'LS'=** |  **_Colour for Line with Syntax Error_**  
---|---  
  
##  Description

Sets numeric colour code used for syntax errors.

Valid colour options for program line number with syntax errors:

|  **0** \- Black |  **4** \- Blue  
---|---|---  
|  **1** \- Red |  **5** \- Magenta  
|  **2** \- Green |  **6** \- Cyan  
|  **3** \- Yellow |  **7** \- White  
  
##  Default

**'LS'=1** (Red)
