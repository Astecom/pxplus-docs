# Mnemonics

**'SB'** |  **_Set Mode to Background_**  
---|---  
  
**Character Display**

##  Description

Use **'SB'** to set background mode. (Characters are displayed at low intensity/dimmed.)

To clear background mode, use the **'CF'** mnemonic.
