# System Parameters  
  
**'BF'=** |  **_Common File Buffers_**  
---|---  
  
##  Description

Assigns the number of common database Keyed I/O buffers to be maintained by the system.

##  Default

**'BF'=10**

#### **Note:**  
When 'BF'=0 (zero, no common buffers), PxPlus uses the setting in the **'FB'=** system parameter.
