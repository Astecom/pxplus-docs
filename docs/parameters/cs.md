# System Parameters

**'CS'** |  **_Coloured Syntax_**  
---|---  
  
##  Description

**_On_** |  Displays program with coloured syntax when using the **LIST** directive and the **LST( )** function.  
---|---  
**_Off_** |  Does not display coloured syntax.  
  
##  Default

**_On_**
