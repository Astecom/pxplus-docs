# Mnemonics

**'EP'** |  **_Start Expanded Print_**  
---|---  
  
**Character Printer**

##  Description

Use '**EP** ' to begin expanded print (either double wide or double high, depending on the type of printer).

To end expanded print, use the **'EF'** mnemonic.
