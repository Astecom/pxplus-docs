# System Parameters

**'AH'** |  **_Alternative 'WINDOW'/'BOX' Heading_**  
---|---  
  
##  Description

Uses alternative **'WINDOW'** /**'BOX'** heading (with the title in inverse video, centered on the top line).

##  Default

**_Off_** \- The title is text, left justified on the top line of the **'WINDOW'** /**'BOX'**.
