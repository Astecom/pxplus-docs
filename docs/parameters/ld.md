# System Parameters

**'LD'** |  **_List Directives in Lowercase_**  
---|---  
  
##  Description

Sets the **LST( )** function and the **LIST** directive to return directive names in lowercase.

##  Default

**_Off_** \- Directives are listed in uppercase.
