# Mnemonics

**'BU'** |  **_Begin Underscoring_**  
---|---  
  
**GUI Display/Printer _or_ Character Display/Printer**

##  Description

Use **'BU'** to begin underscoring characters sent to the display device.

To end underscoring, use the **'EU'** mnemonic.
