# Mnemonics

**'SF'** |  **_Set Mode to Foreground_**  
---|---  
  
**Character Display**

##  Description

Use **'SF'** to set foreground mode. (Characters are displayed at high intensity.)

To clear foreground mode, use the **'CF'** mnemonic.
