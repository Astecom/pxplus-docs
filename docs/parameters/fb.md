# System Parameters

**'FB'=** |  **_Dedicated File Buffers_**  
---|---  
  
##  Description

Assigns the number of dedicated file buffers to use for each file.

#### **Note:**  
PxPlus uses the **'FB'** setting if **'BF'=0** is set (zero, no common buffers).

##  Default

**'FB'=5**
