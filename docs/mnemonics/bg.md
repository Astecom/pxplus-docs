# Mnemonics

**'BG'** |  **_Begin Generating Error #29_**  
---|---  
  
**Behaviour**

##  Description

Use '**BG** ' to begin generating Error #29: Invalid Mnemonic or position specification whenever an invalid mnemonic is executed.

To end generation of Error #29, use the **'EG'** mnemonic.
