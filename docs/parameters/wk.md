# System Parameters

**'WK'** |  **_Keep Window_**  
---|---  
  
##  Description

Prevents **'WINDOW'** and **'DIALOGUE'** boxes from being automatically dropped when you use a **BEGIN** (or **END** in Command mode) or a **CLOSE**.

##  Default

**_Off_**
