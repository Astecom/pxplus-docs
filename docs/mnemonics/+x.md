# Mnemonics

**'+X' & '-X'** |  **_Windows 'X' Close Button_**  
---|---  
  
**GUI Display**

##  Format

_Windows 'X' Close Button On:_**'+X'**  
_  
Windows 'X' Close Button Off:_**'-X'**

##  Description

When the **'+X'** mnemonic is set, use of an **X** icon is set as the default for when new windows are created.
