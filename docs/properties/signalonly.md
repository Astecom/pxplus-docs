# Control Object Properties

**SignalOnly** |  **_Signal Only - Do Not Get Focus_**  
---|---  
  
## Description

Signal only - Do not get focus: 0 = Off; 1 = On

This property can also be read, returning 1 or 0 to indicate if the control is to signal only and not get focus.

## Default

0

## Used By

**BUTTON****CHECK_BOX****RADIO_BUTTON****TRISTATE_BOX**
