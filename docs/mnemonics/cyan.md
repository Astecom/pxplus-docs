# Mnemonics

**'CYAN' & '_CYAN'** |  **_Colour Text_**  
---|---  
  
**GUI Display/Printer _or_ Character Display**

##  Format

_Foreground:_**'CYAN'**  
  
 _Background:_**'_CYAN'**

##  Description

All input or output following this mnemonic will be in Cyan foreground or background.

##  Example

input '_cyan','white',"Please enter your name: ",Name$,
