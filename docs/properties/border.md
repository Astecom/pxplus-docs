# Control Object Properties

**Border** |  **_Add Border_**  
---|---  
  
## Description

Add border: 1 - Border; 0 - No border

#### **Note:**  
Applicable only when using Windows XP or Windows 7 style controls.

## Default

1

## Used By

**CHART****GRID****LIST_BOX****MULTI_LINE****MULTI_LINE** ** _(RTF word processing style)_**
