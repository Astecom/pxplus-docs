# Mnemonics

**'UC'** |  **_Convert Input to Uppercase_**  
---|---  
  
**Editing _or_ Behaviour**

##  Description

Use **'UC'** to convert all subsequent user input to uppercase.

To end uppercase conversion and allow the use of mixed case, use the **'LC'** mnemonic.
