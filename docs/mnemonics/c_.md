# Mnemonics

**'C#'** |  **_Control Cursor Display Mode_**  
---|---  
  
**GUI Display _or_ Character Display**  
  
**_Where:_**

# |  Display mode:  
---|---  
0 |  Hide Cursor |  **'C0'**  
1 |  Display Normal Cursor |  **'C1'**  
2 |  Insert Mode Cursor |  **'C2'**  
  
##  Description

Use the format above (**'C0'** , **'C1'** or **'C2'**) to control how the cursor is displayed.
