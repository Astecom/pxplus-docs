# System Parameters

**'LM'** |  **_Show Matching Search String in List_**  
---|---  
  
##  Description

Highlights matches to a currently defined search string in a LIST operation to the console.

_(The 'LM' system parameter was added in PxPlus v7.)_

##  Default

**_Off_**
