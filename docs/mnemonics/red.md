# Mnemonics

**'RED' & '_RED'** |  **_Colour Text_**  
---|---  
  
**GUI Display/Printer _or_ Character Display**

##  Format

_Foreground:_**'RED'**  
  
 _Background:_**'_RED'**

##  Description

All input or output following this mnemonic will be in Red foreground or background.

input 'red','_white',"Please enter your name: ",Name$,
