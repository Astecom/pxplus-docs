# Mnemonics

**'EG'** |  **_End Generating Error #29_**  
---|---  
  
**Behaviour**

##  Description

Use '**EG** ' to stop generation of Error #29: Invalid Mnemonic or position specification whenever an invalid mnemonic is executed. This is the opposite of the **'BG'** mnemonic.
